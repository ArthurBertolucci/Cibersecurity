# Desafio-2 Bootcamp-Cibersecurity DIO - RANSOMWARE 
### Este desafio consistiu na criação de um ambiente controlado para compreender o funcionamento de um ransomware em nível acadêmico. Foram desenvolvidos scripts que simulam o processo de criptografia de arquivos e a posterior descriptografia, permitindo observar como ocorre a captura de dados e a exibição de mensagens de resgate. Todo o trabalho foi realizado em ambiente de laboratório, sem qualquer aplicação prática fora do contexto educacional. O objetivo foi apenas didático, voltado para entender técnicas de ataque e defesa em cibersegurança, conforme proposto no Bootcamp de Cybersecurity da DIO.

# Desafio-2 Bootcamp-Cibersecurity DIO - KEYLOGGER 
### Este desafio teve como foco a criação de um keylogger acadêmico, com o intuito de compreender como softwares maliciosos podem capturar informações digitadas e enviá-las para diferentes canais, como e-mail e Telegram. O exercício incluiu a configuração de ambiente virtual, uso da biblioteca pynput e testes de execução em terminal Linux. Ressalta-se que todo o desenvolvimento foi feito em ambiente controlado e com finalidade exclusivamente educacional, sem qualquer uso real fora do contexto acadêmico.
